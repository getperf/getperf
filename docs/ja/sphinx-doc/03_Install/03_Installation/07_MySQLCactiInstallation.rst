MySQL, Cacti 設定
======================


Cacti セットアップについて
--------------------------

Getperf 3.1 から Cacti は個別インストールするのではなく、
$GETPERF_HOME/var/cacti の下に Cacti モジュールをバンドルする構成に変更しました。
Cacti を個別インストールする必要はなく、 Cacti のインストールは後述の監視サイト
初期化コマンドで行います。

    .. note:: 

        詳細は、 サイト初期化コマンド :doc:`../10_AdminCommand/01_SiteInitialization` 
        を参照してください。
